# kuliah2/jadual

Visual redesign and benchmark for the Masjid Al-Mukhlisin lecture schedule (`kuliah/jadual/`).

## What this is

A redesigned version of the lecture schedule table. Same live data, improved visual design, and architectural improvements over the production version. Used to test and validate design changes before promoting them to production.

## Access

```
/kuliah2/jadual/jadual.html            — Current month
/kuliah2/jadual/jadual.html?bulan=depan — Next month
/kuliah2/jadual/jadual.html?file=pdf   — PDF export
```

Live at: `dev-data.mamtj6.com/kuliah2/jadual/jadual.html`

## Improvements over production

| Feature | Production (`kuliah/jadual/`) | kuliah2 |
|---------|-------------------------------|---------|
| MINGGU label | CSS `writing-mode` + `transform` (breaks Xibo) | Stacked `<span>` chars, no rotation |
| Arabic text | `Times New Roman` serif | `Scheherazade New` (Google Font) |
| Today highlight | Flat yellow background | Yellow bg + `::after` glow ring |
| Yasiin cell | Standard lecture block | Green cell background via `:has()` |
| XSS safety | Raw string interpolation | `escapeHtml()` everywhere |
| Table borders | `#000` hard black | Softer `#bcbcbc` / `#cfcfcf` |
| Print today cell | Shows yellow highlight | Removed — prints as normal day |
| Offline fallback | Fails silently | `EMBEDDED_DATA` fallback (July 2026) |

## Data source

Reads from `/kuliah/data/jadual_lengkap.json` — the same auto-synced file as production. Do not edit this file manually.
